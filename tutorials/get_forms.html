Get forms for an installation

Once you’ve determined the ETIM Product Class for you installation, you can query 2BA for relevant forms.

Get a list of relevant forms

GET https://apix.2ba.nl/api/V1/Fgo/Forms/All

Returns a list of forms relevant for you installation

Example request:

GET /api/V1/Fgo/Forms/All?classCode=EC010232 HTTP/1.1
Host: apix.2ba.nl
Accept: application/json
Authorization: OAuth token to authenticate

Example response (excerpt):

HTTP/1.1 200 OK
Content-Type: text/javascript

[
  {
      "code": "FGO00002",
      "version": 2,
      "title": "Melding onveilige situatie gas"
  }
]
Request Headers

  • Authorization – OAtuh token to authenticate

Query Parameters

  • classCode (string) – The ETIM Class to filter forms for

Get the form itself

GET https://apix.2ba.nl/api/V1/Fgo/Form/[code]/[version]/

Returns the requested form

Example request:

GET /api/V1/Fgo/Forms/Form/FGO00002/2/ HTTP/1.1
Host: apix.2ba.nl
Accept: application/json
Authorization: OAuth token to authenticate
