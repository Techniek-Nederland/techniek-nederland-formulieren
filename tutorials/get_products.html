Get product information using FGO Plus API¶

In order to easily get all information on a Product, you can query the FGO Plus database. For detailed information on the API, please have a look at the Swagger and the 2BA documentation.

Search for a product¶

GET https://api.fgoplus.nl/api/v1/Fgo/Product/Search¶

Returns a list of products that met your query.

Example request:

GET /api/v1/Fgo/Product/Search?From=0&Size=1&SearchString=nefit&Filters[0].Code=Model&Filters[0].Values[0].Code=ProLine HTTP/1.1
Host: api.fgoplus.nl
Accept: application/json
Authorization: OAuth token to authenticate

Request Headers

  • Authorization – OAuth token to authenticate

Query Parameters

  • From (integer) – Offset to start from (0-based)

  • Size (integer) – Number of results to return

  • SearchString (string) – Filter string

  • Filters (object) – A set of filters. Please see 2BA Swagger for a full list of filters.

Filter fields¶

GET https://api.fgoplus.nl/api/v1/Fgo/Product/FiltersForField¶

Returns a list of models, series and versions that can be used to filter down the list of products.

Example request:

GET /Fgo/Product/FiltersForField?field=Model HTTP/1.1
Host: api.fgoplus.nl
Accept: application/json
Authorization: OAuth token to authenticate

Example response (excerpt):

HTTP/1.1 200 OK
Content-Type: text/javascript

{
  "code": "Model",
  "count": 200,
  "portCode": 0,
  "type": "Model",
  "values": [
      {
          "active": false,
          "code": "Gasgestookte Stralingsverwarmring",
          "count": 40
      },
      ...
  ]
}
Request Headers

  • Authorization – OAuth token to authenticate

Query Parameters

  • Field (string) – The field you want to retrieve filters for. Choices are Category, Brand, Model and Version.

  • Filters (object) –

    A set of filters. Please see 2BA Swagger for a full list of filters.
